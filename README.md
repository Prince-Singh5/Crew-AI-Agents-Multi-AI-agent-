# Crew AI Agents – Multi AI Agent Blog Writer

This project demonstrates how to use **CrewAI agents** with YouTube tools to research and write blog content from videos.  
It sets up two agents — a **Blog Researcher** and a **Blog Writer** — that collaborate to generate blog posts based on YouTube video transcripts.

---

## 🚀 Features
- **Blog Researcher Agent** → Finds relevant YouTube videos and extracts detailed transcription info.  
- **Blog Writer Agent** → Summarizes research into engaging blog content.  
- **YouTube Channel Tool** → Searches videos from a specific channel (`@krishnaik06`).  
- **Task Management** → Defines research and writing tasks with expected outputs.  
- **Environment Variables** → Uses `.env` file for API keys and model configuration.
